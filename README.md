# 世界盃 2026 出線智能計算器

一個單一 HTML 檔嘅世界盃 2026（48 隊・12 組）出線計算工具。輸入每場比分，自動計小組排名、最佳第 3 名同 32 強對陣走線。賽果會自動存喺瀏覽器，下次開返仲喺度。

## 功能

- **分組賽果 / 排名**：12 個小組即時計排名，採用今屆新規則 —— 同分先計**對賽成績**（對賽得分 → 之間得失球 → 之間入球），再到總得失球、總入球。
- **出線情境**：逐個「剩餘比賽勝／和／負」組合精確窮舉，列出邊兩隊出線、邊隊跌第 3；要睇得失球先分到嘅會標示「視得失球」。狀態包括：篤定首名 / 已出線 / 只爭第 3 / 出局。
- **最佳第 3 名**：12 組第 3 名排晒一齊，前 8 名出線，附出線分界分數同每隊差距。
- **32 強走線**：每組首名／次名 ＋ 8 個最佳第 3 名，按 FIFA 官方 **Annex C（全 495 種組合）** 自動配對。撳隊名揀贏方，一路帶上去 16 強 → 8 強 → 4 強 → 決賽（連季軍戰）。

## 用法

直接用瀏覽器打開 `index.html` 即可。國旗圖需要連網先顯示。
