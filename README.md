# 世界盃 2026 出線智能計算器

一個單一 HTML 檔嘅世界盃 2026（48 隊・12 組）出線計算工具。輸入每場比分，自動計小組排名、最佳第 3 名同 32 強對陣走線。賽果會自動存喺瀏覽器，下次開返仲喺度。

## 功能

- **分組賽果 / 排名**：12 個小組即時計排名，採用今屆新規則 —— 同分先計**對賽成績**（對賽得分 → 之間得失球 → 之間入球），再到總得失球、總入球。
- **出線情境**：逐個「剩餘比賽勝／和／負」組合精確窮舉，列出邊兩隊出線、邊隊跌第 3；要睇得失球先分到嘅會標示「視得失球」。狀態包括：篤定首名 / 已出線 / 只爭第 3 / 出局。
- **最佳第 3 名**：12 組第 3 名排晒一齊，前 8 名出線，附出線分界分數同每隊差距。
- **32 強走線**：每組首名／次名 ＋ 8 個最佳第 3 名，按 FIFA 官方 **Annex C（全 495 種組合）** 自動配對。撳隊名揀贏方，一路帶上去 16 強 → 8 強 → 4 強 → 決賽（連季軍戰）。

## 用法

直接用瀏覽器打開 `index.html`（會自動跳去主程式）即可。國旗圖需要連網先顯示。

## 輸入賽果

人手輸入每場比分，賽果自動存喺瀏覽器（localStorage）。更新 HTML 重新上載唔會清走你嘅資料（同一網址／瀏覽器）。

## 試算模式

撳 **「🧪 試算模式」**（或撳「填入示範賽果」會自動入試算模式）：喺度點改都**唔會儲存**，試完撳「↩ 退出試算」即刻還原你嘅真實賽果。

## 匯出 / 匯入備份

- **匯出備份**：下載一個 `wc2026-data.json`，做備份或者搬去第二部機／瀏覽器。
- **匯入**：揀返個 json 檔還原。

## 公開顯示賽果（畀人開網址都見到）

因為 localStorage 淨係你自己部機見到。想公開網址人人見到最新賽果：撳「匯出備份」→ 將檔案改名做 **`data.json`** → 連 `index.html`、主程式一齊 commit 上 GitHub。網頁開頁時（本機未有資料）會自動讀 `data.json` 做預設顯示。每次想更新公開版，就再匯出 + 覆蓋 `data.json`。

## 部署到 GitHub Pages

上載到 repo：`index.html`（跳轉頁）、`世界盃2026出線計算器.html`（主程式），想公開賽果就連 `data.json`。再開 Settings → Pages → main / root。`commentary.db`、`reference.db` 等私人資料檔**唔好上載**。
